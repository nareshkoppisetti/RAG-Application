Oracle SSO is a common phrase that refers enabling Single Sign on for Oracle Applications, also known as Oracle Financials, Oracle E-Business Suite(EBS), or simply Oracle ERP. Oracle Applications is generally referred as Oracle in the IT industry.

Oracle EBS had been traditionally integrated with Oracle 10g Application Server (Oracle 10gAS) Single Sign On (OSSO 10g)

OSSO 10g has been replaced by Oracle Access Manager(OAM 10g, 11g, and 12c ), which was originally Oblix CoreID acquired by Oracle.

Oracle Identity Cloud Services (IDCS) is the modern cloud SSO solution offered by Oracle Cloud.

SSOGEN is a third party product, certified by Oracle EBS team, and it serves as an alternative to Oracle 10gAS SSO, OSSO 10g, OAM, OID, and IDCS.
